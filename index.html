Dropdown roles

Get your own menu roles for your discord server
written in discord.js v13 by Fruity

Getting started

Download the source code here and enter details in config.json

{
    "token":"", //bot token
    "r1":"", //role1id
    "r2":"", //role2id
    "r3":"", //role3id
    "r4":"", //role4id
    "r5":"", //role5id
    "r1m":"", //role 1 emoji
    "r2m":"", //role 2 emoji
    "r3m":"", //role 3 emoji
    "r4m":"", //role 4 emoji
    "r5m":"", //role 5 emoji

}

Install

npm i

Start

node.
